# Política de Privacidade - Cosmic Vision

**Última atualização:** 25 de dezembro de 2025

## Introdução

Esta Política de Privacidade descreve como o aplicativo **Cosmic Vision** ("nós", "nosso" ou "aplicativo") coleta, usa e protege as informações dos usuários ("você" ou "usuário"). Ao usar o Cosmic Vision, você concorda com as práticas descritas nesta política.

O Cosmic Vision é um aplicativo gratuito desenvolvido por **Marcus Paixão** que permite aos usuários explorar imagens astronômicas fornecidas pela NASA através da API APOD (Astronomy Picture of the Day).

## Informações que Coletamos

### Informações que NÃO coletamos:

- **Dados pessoais:** Não coletamos nome, e-mail, telefone ou qualquer informação de identificação pessoal.
- **Dados de localização:** Não acessamos ou armazenamos sua localização.
- **Dados de contatos:** Não acessamos sua lista de contatos.
- **Dados de login:** Não exigimos criação de conta ou login.
- **Dados financeiros:** Não processamos pagamentos ou dados financeiros.

### Informações armazenadas localmente no dispositivo:

O aplicativo armazena as seguintes informações **apenas no seu dispositivo** (armazenamento local):

- **Favoritos:** Lista de imagens astronômicas que você marcou como favoritas.
- **Preferências:** Configurações do aplicativo como tema, qualidade de imagem preferida e configurações de notificação.
- **Cache de imagens:** Imagens temporárias para melhorar a performance e reduzir uso de dados.

Estas informações são armazenadas localmente usando SQLite e SharedPreferences, e **nunca são enviadas para servidores externos**.

## Permissões do Aplicativo

### Android

- **Internet (INTERNET):** Necessária para buscar imagens e informações da API NASA APOD.
- **Armazenamento (WRITE_EXTERNAL_STORAGE):** Utilizada apenas em versões antigas do Android (abaixo do Android 10) para salvar imagens na galeria quando solicitado pelo usuário.

### iOS

- **Acesso à Galeria de Fotos:** Solicitada apenas quando você escolhe salvar uma imagem na galeria do dispositivo.

## Uso de Dados de Terceiros

### API NASA APOD

O Cosmic Vision utiliza a API pública da NASA (api.nasa.gov) para obter imagens astronômicas. Quando você usa o aplicativo:

- Requisições são enviadas aos servidores da NASA para obter as imagens do dia.
- A NASA pode coletar informações técnicas básicas conforme sua própria política de privacidade.
- Recomendamos consultar a [Política de Privacidade da NASA](https://www.nasa.gov/about/highlights/HP_Privacy.html) para mais detalhes.

### Análise e Publicidade

- **Não utilizamos** serviços de análise (analytics) de terceiros.
- **Não exibimos** anúncios no aplicativo.
- **Não rastreamos** seu comportamento de uso.

## Compartilhamento de Dados

**Não compartilhamos** nenhuma informação com terceiros. Os únicos dados transmitidos são as requisições técnicas necessárias para buscar conteúdo da API da NASA.

## Segurança dos Dados

Implementamos medidas de segurança para proteger as informações armazenadas localmente:

- Os dados são armazenados apenas no seu dispositivo.
- Utilizamos conexões HTTPS seguras para todas as comunicações com a API da NASA.
- Não mantemos servidores próprios que armazenem dados de usuários.

## Dados de Crianças

O Cosmic Vision não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente informações de crianças. Se você é pai ou responsável e acredita que seu filho nos forneceu informações pessoais, entre em contato conosco.

## Seus Direitos

Você tem total controle sobre seus dados locais:

- **Acesso:** Todas as informações estão visíveis no próprio aplicativo (favoritos e configurações).
- **Exclusão:** Você pode limpar o cache e remover favoritos a qualquer momento nas configurações do app.
- **Desinstalação:** Ao desinstalar o aplicativo, todos os dados locais são automaticamente removidos.

## Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas através de uma atualização no aplicativo. A data da última atualização será sempre indicada no início deste documento.
